Snake Game (C++)

A simple command-line implementation of the classic Snake game written in **C++**.  
The project was built to practice object-oriented programming, modular design (splitting code into multiple `.h` and `.cpp` files), and handling real-time user input.

 Features
- Classic snake movement (up, down, left, right)
- Food system that increases snake length
- Collision detection (walls and self)
- Modular codebase split into headers and source files
- Runs directly in the terminal

 Technologies
- C++ Version 17.14.14
- Standard Template Library (STL)
- Cross-platform (tested on Windows, Linux, macOS with minor tweaks)
